# utilisation-forEach-
Simple JavaScript app that adds user input to an array and displays the updated list dynamically using forEach() and DOM manipulation.
# JavaScript forEach() Practice Project

A small JavaScript practice project that demonstrates how to:

- Add user input into an array
- Display array content dynamically
- Use the `forEach()` method
- Manipulate the DOM with JavaScript
- Create and append HTML elements dynamically
- Validate empty input values

## Features

- Add words into an array
- Display the array content in real time
- Generate list items dynamically with `forEach()`
- Prevent empty input submission
- Responsive text wrapping for long arrays

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## What I Learned

Through this project, I practiced:

- DOM selection and manipulation
- Event listeners
- Array methods (`push`, `forEach`, `join`)
- Input validation
- Dynamic element creation with `createElement()`
- Updating the UI dynamically

## How It Works

1- The user enters a word in the input field.
2- Clicking the button adds the word into an array.
3- The array content is displayed on the page.
4-Each item is also displayed inside a dynamic HTML list.
